# 🌐 Linktree - Junior Dev

Este projeto é uma página estilo Linktree desenvolvida para centralizar meus principais links, redes sociais e projetos em um único lugar.

## 🚀 Acesse o projeto

👉 https://juniordevti.github.io/meusitepessoal/

---

## ✨ Funcionalidades

- 🔗 Centralização de links (Website, Instagram, YouTube, Spotify, LinkedIn, Discord e GitHub)
- 🎨 Interface moderna e responsiva
- 🌈 Background com gradiente animado
- 👤 Foto de perfil com animação
- ⚡ Efeitos de hover nos botões
- 💎 Estilo visual inspirado em apps modernos

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em:

- 📱 Mobile
- 💻 Desktop
- 📟 Tablets

---

## 🎯 Objetivo

Criar uma página simples, bonita e funcional para apresentar meus links de forma organizada e profissional.

---

## 📌 Melhorias futuras

- 🎵 Integração com player do Spotify
- ✨ Animações mais avançadas
- 📊 Analytics de cliques
